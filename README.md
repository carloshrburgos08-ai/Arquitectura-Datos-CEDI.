# Portafolio Analítico: Operaciones, Producción Textil y Logística B2B 🏭📦

Este repositorio documenta mi enfoque técnico y estratégico para la gestión integral de la cadena de suministro en modelos de **Paquete Completo (Full Package)** y maquila, con más de 10 años de experiencia en el sector.

## 📄 Documentación Técnica Incluida

En este documento puedes consultar el **Informe de Optimización de Operaciones y Arquitectura Logística** (archivo PDF), en el cual integro:

1. **Arquitectura de Datos Logísticos:** Estructuras relacionales diseñadas en SQL para el seguimiento estricto de despachos a CEDI y control de inventarios.
2. **Diagramas de Flujo Lean Manufacturing:** Mapeo de procesos en líneas de confección enfocados en la reducción de Lead Time y aplicación de metodologías *Just-in-Time*.
3. **Control de Gestión (KPIs):** Definición de métricas clave como OTD (On Time Delivery), eficiencia operativa y control de costo objetivo.
